# Glossary

Programming paradigm
A style of programming. Main paradigms: procedural, object oriented, functional

Object oriented programming (OOP)
A style of programming that couples functions with the data they need in a objects.

Class
The main object in OOP. Classes are defined to encapsulate data and the routines that operate on that data.

Instantiation
The process of creating an object from a class. Class constructor invokes the class's special initialiser. Usually used to set initital state of an instance.

Instance
The object that results from instantiation.

Instance variables
The data stored as properties on the instance, using self. They are not global variables. They are accessible from any subroutine in the instance that can access self.

Instance methods
The subroutines defined within the class. They are not global subroutines. They are accessible from any subroutine in the instance that can access self.
